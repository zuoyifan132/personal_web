项目展示

agents_collaboration

This script is designed to simulate an agent-based chat environment, specifically tailored for idea innovation generation tasks. Multiple agents interact according to specified roles and attributes, making it ideal for exploring and generating innovative ideas in a controlled setting.

autogen

AutoGen is an open-source programming framework for building AI agents and facilitating cooperation among multiple agents to solve tasks. AutoGen aims to streamline the development and research of agentic AI, much like PyTorch does for Deep Learning. It offers features such as agents capable of interacting with each other, facilitates the use of various large language models (LLMs) and tool use support, autonomous and human-in-the-loop workflows, and multi-agent conversation patterns.

DPO_LLMEvaluate

Large-scale unsupervised language models (LMs) offer broad knowledge and reasoning capacities but lack precise behavioral control due to their unsupervised nature. Current fine-tuning methods with reinforcement learning from human feedback (RLHF) are complex and often unstable. In this report, we propose using Direct Preference Optimization (DPO), a stable, efficient, and performant algorithm, to train a compact LM that generates succinct, relevant, and context-aware dialogues. By simplifying the alignment of LMs with human preferences through a single stage of policy training, DPO eliminates the need for fitting a reward model or extensive hyperparameter tuning. The anticipated outcome of our study is to develop a conversational partner that ensures smooth, intuitive, and naturally-flowing dialogues, resonating with human conversational norms. The Experiment Result shows that GPT2-XL achieves similar performance with OPT1.3B after DPO process and Original GPT2-XL increase in accuracy 7.1 and overall score 10.3 post-DPO

InstructGPT_finetuning

This project implements a reinforcement learning-based dialog agent using the Proximal Policy Optimization (PPO) algorithm. The agent generates responses to given input sequences using a pre-trained GPT-2 language model and is trained using PPO to maximize its expected reward.

Route_Planning_with_DQN

This repository contains a Deep Q-Network (DQN) implementation for solving Multi-Agent Path Finding (MAPF) problems.

Linux_LLM_Tools_System

The Linux LLM Tools System is a command-line tool designed to interact with a Linux environment by processing user commands in natural language. It uses an AI-powered model (such as OpenAI's API) to classify and execute various system operations like file, directory, and web operations. The system can run continuously to process user requests or handle one-time programmatic operations based on user inputs.
